== Description ==

Ajax SearchWP is a WordPress plugin that enhances your website's search functionality with live search results and autocomplete features. It allows users to search across various content types including posts, pages, custom post types, and WooCommerce products.

== Features ==

* Live search results using AJAX technology.
* Set a limit on the number of results in AJAX searches.
* Update the search input placeholder text.
* Customize the "No results found" message.
* Autocomplete search queries based on user input and previous searches.
* Option to select custom post types to include in the search.
* Seamless integration with popular themes and page builders like Elementor and Divi.
* Lightweight and optimized for performance.
* Shortcode for easy insertion of the search form.
* Compatible with WordPress 5.0 and higher.


== Installation ==

1. Upload the `ajax-searchwp` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to 'Settings' -> 'Ajax SearchWP' to configure the plugin.
4. Use the shortcode `[ajax_searchwp]` to display the search form.

== Frequently Asked Questions ==

= How do I use the shortcode? =
Use the shortcode `[ajax_searchwp]` in any post or page where you want to display the search form.

= Can I customize the post types included in the search? =
Yes, you can select the post types you want to include in the search from the plugin settings page.
